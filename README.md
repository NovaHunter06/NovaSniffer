# NovaSniffer v3.2 🛡️
**Author:** Rohan Das (@NovaHunter06)

NovaSniffer is a professional-grade Network Intelligence & Forensics tool. It provides real-time traffic analysis, credential sniffing, and Geo-IP/WHOIS enrichment via a modern web-based dashboard.

## 🚀 Features
- **Real-time Traffic Stream**: Monitor TCP, UDP, and ICMP traffic.
- **Security Alerts**: Automatic detection of cleartext credentials.
- **Intelligence Enrichment**: Built-in Geo-IP and ISP identification.
- **Forensics Ready**: Simultaneous binary (.pcap) and text (.txt) logging.
